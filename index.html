Немного о C++: (нажмите, чтобы показать)

C++ (си-плюс-плюс) — это язык программирования общего назначения, который сочетает возможности низкоуровневого программирования (как в языке Си) с высокоуровневыми абстракциями. Простыми словами, на C++ можно создавать программы любой сложности: от простых консольных утилит до сложных игровых движков.

Язык создан датским программистом Бьёрном Страуструпом в начале 1980-х годов. Сначала Страуструп называл язык «C with classes» («Си с классами»). В 1983 году язык переименован в C++. Первый коммерческий выпуск состоялся в октябре 1985 года. В 1998 году опубликован первый международный стандарт C++ (ISO/IEC 14882), который установил основные правила и рекомендации по использованию языка.

Особенности: (нажмите, чтобы показать)

  • Мультипарадигменность — поддержка нескольких парадигм программирования.
  • Производительность — возможность писать высокооптимизированный код для задач, требующих высокой производительности (например, игры, симуляции, научные вычисления).
  • Прямой доступ к памяти — низкоуровневый доступ через указатели, что даёт разработчикам полный контроль над ресурсами.
  • Стандартная библиотека (STL) — предоставляет контейнеры (например, vector, map), алгоритмы (например, сортировка, поиск) и утилиты (например, умные указатели).
  • Кроссплатформенность — программы на C++ могут быть скомпилированы и запущены на различных платформах (Windows, Linux, macOS и др.).

Применение (нажмите, чтобы показать)

C++ используется в разных областях, например:

  • Системное программирование — операционные системы, драйверы.
  • Игры — игровые движки (Unreal Engine, CryEngine).
  • Научные вычисления — моделирование, симуляции.
  • Финансовые технологии — высокочастотные торговые системы.
  • Встроенные системы — микроконтроллеры, IoT.

Пространства имён (namespace) в C++

Пространство имён в C++ (англ. namespace) — это логический контейнер для имен (функций, классов, переменных), который помогает избежать конфликтов имен в больших проектах, особенно при использовании разных библиотек, и организовать код по функциональным блокам. Для доступа к элементам внутри пространства имён используется оператор разрешения области видимости ::, например, std::cout.

Зачем нужны пространства имён?

Предотвращение конфликтов имен:

Если у вас есть две разные библиотеки, и обе содержат функцию с одинаковым названием, например, print(), то в одном проекте возникнет конфликт, так как компилятор не будет знать, какую из них использовать. Пространство имён решает эту проблему, позволяя задать уникальные имена функциям, даже если они называются одинаково в разных контейнерах.

Структурирование кода:

Пространства имён помогают логически группировать взаимосвязанные компоненты, создавая более организованную и поддерживаемую структуру проекта.

Удобство работы с библиотеками:

Стандартная библиотека C++ находится в пространстве имён std. Чтобы использовать её элементы, например, cout или string, вы должны указать std:: перед их именем, чтобы сообщить компилятору, из какого именно пространства имён взять нужный элемент. Как использовать пространства имён?

Объявление своего пространства имён:


    namespace my_namespace {

        }
    }

    std::cout << "Привет";       
    //Вызов функции из пространства std

Использование директивы using:

Классы, структуры, объекты (ООП) в C++

Что такое ООП в C++?

Объектно-ориентированное программирование (ООП) — это подход, при котором программа рассматривается как набор объектов(классов, структур),взаимодействующих друг с другом.

Классы

Классы в C++ — это пользовательские типы данных, объединяющие переменные (свойства) и функции (методы) в единую структуру для создания объектов. Они являются основой объектно-ориентированного программирования (ООП), позволяя моделировать реальные сущности как объекты с их данными и поведением. Основные компоненты класса — это свойства (данные) и методы (функции), которые работают с этими данными.

пример:

class Car {
private: //этот раздел доступен только в классе 
    string brand;
    int year;
    //это Поле доступно для изменения только разработчику
public:// этот раздел доступен извне класса
    // Конструктор
    Car(string b, int y) {
        brand = b;  //тут мы устанавливаем данные в приватные поля
        year = y; //и тут, позже эти поля можно менять только так
        // напрямую нельзя
    }
    
    // Метод, это функция для работы с классами
    void displayInfo() {
        cout << "Марка: " << brand << ", Год: " << year << endl;
    }
};

// Создание объекта
Car myCar("Toyota", 2020); //в string запишется toyota, а в int - 2020
myCar.displayInfo();

[if, else] условия.



Условие, или же if-else в C++ позволяет выполнять
разные блоки кода в зависимости от истинности
некоторого условия. Его структура следующая:
if (условие) { // код, если условие истинно }
else { // код, если условие ложно }. Можно
использовать и другие варианты, такие как
if без else, или добавить сколько угодно блоков else if для проверки нескольких условий
последовательно. 



пример:


int b = 3;
if (b == 5) {
    std::cout << "b равно 5" << std::endl;
} else {
    std::cout << "b не равно 5" << std::endl; // Выведет "b не равно 5"
}


Конструкция if, else, if-else.
Это позволяет проверять несколько условий
поочередно. Если первое условие (if) ложно,
проверяется второе условие (else if, его кстати можно писать сколько угодно раз), и так
далее. Если ни одно из условий не истинно,
выполняется блок else. 



пример:


int num = 7;
if (num < 5) {
    std::cout << "Число меньше 5" << std::endl;
} else if (num == 7) {
    std::cout << "Число равно 7" << std::endl; // Выведет "Число равно 7"
} else {
    std::cout << "Число больше 5, но не равно 7" << std::endl;
}

[for, do, while] циклы.



В C++ циклы for, while и do-while используются
для повторения выполнения блоков кода. for
подходит для выполнения кода заданное
количество раз, while выполняет код, пока
условие истинно (проверяет условие до
выполнения тела), а do-while
гарантирует выполнение кода как
минимум один раз, проверяя условие
после выполнения тела. 

            
Цикл for используется, когда количество
итераций известно заранее. 


пример:


for (int i = 0; i < 5; ++i ) {
    // int i = 0 создаём переменную для того чтобы цикл работал 
    // i < 5 условие когда цикл завершится
    // ++i после каждой итерации цикла
     значение i увеличивается на заданное значение и цикл повторяется 
}

Цикл while (с предусловием) выполняет код до
тех пор, пока условие истинно. 


пример:


int count = 0;
while (count < 3) {
    // этот код выполнится 3 раза
    count++;
}

Цикл do-while (с постусловием) выполняет тело
цикла один раз, а затем повторяет его, пока
условие истинно. 




пример:


int num = 0;
do {
    // этот код выполнится как минимум один раз
    num++;
} while (num < 3); //будет выполняться пока значение в скобках 
//истинно

[switch, case] ещё один вид условия


Оператор switch-case в C++ — это множественный
условный оператор, который позволяет сравнить
одно выражение с набором разных значений и
выполнить соответствующий блок кода. Он
имеет следующую структуру: switch(выражение)
{ case значение: выполнение блока кода;
break; ... default: выполнение блока, если
совпадений нет break; }. Важно использовать оператор break для выхода
из блока case, иначе выполнение
продолжится в следующем блоке. 


Как это работает?
1.   Сравнение выражения: Программа оценивает
     выражение в скобках switch().

2.   Поиск совпадения: Значение выражения
     последовательно сравнивается с константными
     значениями, указанными после каждого
     оператора case.

3.   Выполнение блока: Если найдено
     совпадение, выполняется блок кода,    следующий    за этим case.
     Оператор break: После выполнения блока кода в
     case необходимо использовать оператор break.
     Без него выполнение перейдет к следующему
     case и будет продолжаться до первого break или
     до конца конструкции switch.

4.   Блок default: Если ни одно из значений в
     case не совпало с выражением, выполняется
     необязательный блок default. 
     
     
пример:


#include 

int main() {
    int day = 3;
    std::string dayName;

    switch (day) {
        case 1:
            dayName = "Понедельник";
            break; // Выход из switch
        case 2:
            dayName = "Вторник";
            break;
        case 3:
            dayName = "Среда";
            break;
        case 4:
            dayName = "Четверг";
            break;
        case 5:
            dayName = "Пятница";
            break;
        case 6:
            dayName = "Суббота";
            break;
        case 7:
            dayName = "Воскресенье";
            break;
        default:
            dayName = "Некорректный день"; // Если day не 1-7
            break;
    }

    std::cout << "Это " << dayName << std::endl; // Вывод: Это Среда

    return 0;
}

ключевое слово [const]


В C++ const — это ключевое слово, которое
сообщает компилятору и другим разработчикам,
что значение переменной, объекта или ссылки не
может быть изменено после инициализации.
Использование const повышает безопасность
кода, предотвращая случайные изменения данных,
улучшает читаемость программы и дает
компилятору возможность для оптимизации. 

Объявление переменной с const делает ее
 неизменяемой. Попытка присвоить новое
  значение такой переменной приведет к ошибке
   во время компиляции.


пример:


const int PI = 3; // Инициализация константы
// PI = 4; // Ошибка: нельзя изменять константу

[try, catch]


Конструкция try-catch в C++ — это механизм
обработки исключений, который позволяет
перехватывать и обрабатывать ошибки,
возникающие в коде. Блок try содержит код,
который может вызвать исключение, а блок
catch обрабатывает пойманное исключение. Это
позволяет программе корректно реагировать на
исключительные ситуации, а не аварийно завершаться. 


Как это работает???


Блок try: Внутри этого блока помещается код,
выполнение которого может привести к ошибке
(например, деление на ноль, некорректные входные данные).


Ключевое слово throw: Если в блоке try
происходит ошибка, возбуждается исключение с
помощью ключевого слова throw.
  

Блок catch: Как только исключение выброшено,
выполнение блока try немедленно прекращается,
и управление передаётся в блок catch.

Обработка исключения: В блоке catch
выполняется код, который обрабатывает
пойманную ошибку. После выполнения кода в
catch, управление передается следующей
инструкции после всего блока try-catch.

Несколько блоков catch: Можно использовать
несколько блоков catch для обработки разных
типов исключений. Выбирается тот блок catch,
тип параметра которого соответствует типу
выброшенного исключения. 
    


пример:


#include 

int main() {
    int result = dividend / divisor;
    int dividend = 10;
    int divisor = 0; // Ситуация, которая вызовет ошибку

    try {
        if (divisor == 0) {
            throw "Деление на ноль!"; // Возбуждаем исключение
        }
        std::cout << "Результат: " << result << std::endl;
    } catch (const char* error_message) { // Ловим исключение типа const char*
        std::cerr << "Ошибка: " << error_message << std::endl; // Обрабатываем ошибку
    }

    return 0;
}
если вы обратили внимание что в коде
есть какое-то throw то:

В C++ throw — это ключевое слово, используемое
для генерации исключения при возникновении
ошибки или непредвиденной ситуации в
программе. Оператор throw остановлет
выполнение текущей функции, передавая
управление в блок catch в стеке вызовов, где
происходит обработка ошибки. Вместе с try и
catch оператор throw является частью механизма
обработки исключений, позволяющего
эффективно управлять ошибками во время
выполнения программы.

[new, delete]


new и delete — это операторы в C++,
используемые для динамического выделения и
освобождения памяти в куче соответственно.
new выделяет память и создает объект,
возвращая указатель(*) на него, а delete
освобождает эту память, вызывая деструктор
объекта перед освобождением. Для массивов
используются соответствующие операторы new[] и delete[]. 

если что указатель это эта звёздочка (*)
она очень полезна в больших программах
так как при её использовании на переменную
 если начать работу с этой переменной то
  память для неё не будет выделяться. это
   оптимизирует программу.
   

пример:


int* ptr = new int; // Выделяет память для одного int
delete ptr; // Освобождает память, на которую указывает ptr


int* arr = new int[10]; // Выделяет память для массива из 10 int'ов
delete[] arr; // Освобождает память для массива, на который указывает arr
